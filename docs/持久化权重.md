 我希能持久化权重应该怎么做，比如有一组参数效果好，我希望保留这组参数的权重

可以，用 PyTorch 的标准做法是保存 state_dict。你这里有两个模型：

encoder
decoder

所以要把二者的权重都保存下来。一般还会顺便保存一些训练配置，比如：

hidden_size
vocab_size
stoi
itos
epoch
loss
accuracy

这样以后恢复时不容易搞混。

---
1. 保存权重

训练完之后执行：

checkpoint = {
    "encoder_state_dict": encoder.state_dict(),
    "decoder_state_dict": decoder.state_dict(),
    "hidden_size": hidden_size,
    "vocab_size": vocab_size,
    "stoi": stoi,
    "itos": itos,
}

torch.save(checkpoint, "date2date_seq2seq_rnn.pt")

这样会在当前目录生成一个文件：

date2date_seq2seq_rnn.pt

这个文件就是持久化后的权重和配置。

---
2. 恢复权重

以后重新打开 notebook，只要先执行前面的定义：

1. import torch
2. stoi / itos / vocab_size
3. EncoderRNN
4. DecoderRNN

然后加载：

checkpoint = torch.load("date2date_seq2seq_rnn.pt")

hidden_size = checkpoint["hidden_size"]
vocab_size = checkpoint["vocab_size"]

encoder = EncoderRNN(vocab_size, hidden_size)
decoder = DecoderRNN(vocab_size, hidden_size, vocab_size)

encoder.load_state_dict(checkpoint["encoder_state_dict"])
decoder.load_state_dict(checkpoint["decoder_state_dict"])

encoder.eval()
decoder.eval()

然后就可以直接：

generate("2002-1-23", encoder, decoder)

---
3. 为什么要重新创建模型再加载？

因为 state_dict 只保存参数，不保存模型结构本身。

所以加载时必须先有同样结构的模型：

encoder = EncoderRNN(vocab_size, hidden_size)
decoder = DecoderRNN(vocab_size, hidden_size, vocab_size)

再把权重塞进去：

encoder.load_state_dict(...)
decoder.load_state_dict(...)

如果你保存时用的是：

hidden_size = 128

加载时也必须创建：

hidden_size = 128

否则 shape 对不上。

---
4. 推荐你封装成两个函数

可以在 notebook 后面加两个函数。

保存函数

def save_checkpoint(path, encoder, decoder, hidden_size, extra=None):
    checkpoint = {
        "encoder_state_dict": encoder.state_dict(),
        "decoder_state_dict": decoder.state_dict(),
        "hidden_size": hidden_size,
        "vocab_size": vocab_size,
        "stoi": stoi,
        "itos": itos,
    }

    if extra is not None:
        checkpoint["extra"] = extra

    torch.save(checkpoint, path)

调用：

save_checkpoint(
    "date2date_seq2seq_rnn.pt",
    encoder,
    decoder,
    hidden_size,
    extra={
        "sample_size": sample_size,
        "epochs": 10,
        "lr": 0.001,
        "final_loss": avgloss,
    },
)

---
加载函数

def load_checkpoint(path):
    checkpoint = torch.load(path)

    hidden_size = checkpoint["hidden_size"]
    vocab_size = checkpoint["vocab_size"]

    encoder = EncoderRNN(vocab_size, hidden_size)
    decoder = DecoderRNN(vocab_size, hidden_size, vocab_size)

    encoder.load_state_dict(checkpoint["encoder_state_dict"])
    decoder.load_state_dict(checkpoint["decoder_state_dict"])

    encoder.eval()
    decoder.eval()

    return encoder, decoder, checkpoint

调用：

encoder, decoder, checkpoint = load_checkpoint("date2date_seq2seq_rnn.pt")

print(checkpoint["extra"])
print(generate("2002-1-23", encoder, decoder))

---
5. 如果还想继续训练，要不要保存 optimizer？

如果你只是想“保存一个效果好的模型，用来 generate”，保存 encoder/decoder 就够了。

但如果你希望以后从这个点继续训练，建议把 optimizer 也保存：

checkpoint = {
    "encoder_state_dict": encoder.state_dict(),
    "decoder_state_dict": decoder.state_dict(),
    "encoder_optimizer_state_dict": encoder_optimizer.state_dict(),
    "decoder_optimizer_state_dict": decoder_optimizer.state_dict(),
    "hidden_size": hidden_size,
    "vocab_size": vocab_size,
    "stoi": stoi,
    "itos": itos,
}

加载时：

encoder_optimizer = torch.optim.Adam(encoder.parameters(), lr=0.001)
decoder_optimizer = torch.optim.Adam(decoder.parameters(), lr=0.001)

encoder_optimizer.load_state_dict(checkpoint["encoder_optimizer_state_dict"])
decoder_optimizer.load_state_dict(checkpoint["decoder_optimizer_state_dict"])

这样可以保留 Adam 里的动量等内部状态，继续训练会更平滑。

---
6. 训练和推理模式也要注意

保存前通常无所谓，但推理时建议：

encoder.eval()
decoder.eval()

因为你的 EncoderRNN 里有：

self.dropout = nn.Dropout(0.1)

训练时 dropout 会随机丢弃一部分 embedding；推理时应该关闭随机性。

如果要继续训练，再切回来：

encoder.train()
decoder.train()

---
7. 推荐命名方式

如果你会做很多组实验，建议文件名里带上关键信息：

checkpoints/rnn_h128_lr001_samples5000_epochs10_acc036.pt

例如：

path = "checkpoints/rnn_h128_lr001_samples5000_epochs10.pt"
save_checkpoint(path, encoder, decoder, hidden_size)

如果目录不存在，可以先：

from pathlib import Path

Path("checkpoints").mkdir(exist_ok=True)

---
8. 最小可用版本

如果你现在只想立刻保存当前这组权重，就直接加这一段：

torch.save(
    {
        "encoder_state_dict": encoder.state_dict(),
        "decoder_state_dict": decoder.state_dict(),
        "hidden_size": hidden_size,
        "vocab_size": vocab_size,
    },
    "date2date_seq2seq_rnn.pt",
)

以后恢复：

checkpoint = torch.load("date2date_seq2seq_rnn.pt")

encoder = EncoderRNN(checkpoint["vocab_size"], checkpoint["hidden_size"])
decoder = DecoderRNN(checkpoint["vocab_size"], checkpoint["hidden_size"], checkpoint["vocab_size"])

encoder.load_state_dict(checkpoint["encoder_state_dict"])
decoder.load_state_dict(checkpoint["decoder_state_dict"])

encoder.eval()
decoder.eval()

然后：

generate("2002-1-23", encoder, decoder)

一句话总结：

用 torch.save 保存 encoder.state_dict() 和 decoder.state_dict()；
用 torch.load 读取后，先创建同结构模型，再 load_state_dict 恢复权重。
